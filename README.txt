This repository shows a hands-on project for a finance tool.
The goal is to demonstrate both the theoretical and practical knowledge involved in the full IT process of building and deploying a website.

It walks through the step-by-step work on the frontend, backend, database, security, and cloud + DevOps.


Este repositório apresenta um projeto prático de uma ferramenta de finanças.
Seu objetivo é demonstrar os conhecimentos teóricos e práticos envolvidos em todo o processo realizado na área de Tecnologia da Informação para a implementação de um site.

O projeto apresenta, de forma passo a passo, as etapas de desenvolvimento de frontend, backend, banco de dados, segurança, além de cloud e DevOps.
